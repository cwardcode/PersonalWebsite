CS350 - Intro to Computer Organization

Project 1

ReverseAdd

A MIPS application that reads a string composed of four numeric chars. Once read the application prints the string as entered, starting with the most significant bit. After printing, the application adds every integer and prints the corresponding value. After it has been added, print the original string in reverse.

Project 2

Spaced_Insertion_Sort

A MIPS application that sorts a list of arrays by implementing an Insertion Sort algorithm. This application introduced the concept of MIPS procedure calls, and creating procedure calls based upon standard conventions

Project 3

Shell Sort

A MIPS application that implements a shell procedure along with the Spaced Insertion Sort procedure from the prior project. This introduced another concept of MIPS coding -- allocating data to the stack. For each array that was to be sorted, it had to be saved to the stack when the application loads.

Project 4

Cache Simulator

An application that mimics an n-way cache. Providing the application an input file containing the cache configuration, followed by hexadecimal memory addresses will result in the program displaying a "cache trace", which displays the address, tag, index, offset, and times referenced, along with hit/miss ratio statistics.
