CS453 - Database Systems

Project 1

E-Commerce Website

This course consisted of a semester-long project to develop an e-commerce website, powered by the python-based Django Framework. All customer data, retail location data, and items were stored in an SQL backend. The store featured a customer reward system, searchable maps to find retail locations based upon addresses entered.

Course Description

Relational query languages (SQL), database design, normal forms, storage structures, access methods, query processing, crash recovery, transactions, distributed databases, object-oriented and multimedia databases.
