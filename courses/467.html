CS467 - Mobile Application Development

Project 1

Basic calculator

The first assignment in this course was to make a simple calculator for Android. The application required basic calculator functionality, including addition, subtraction, multiplication, and division. The application also required both a horizontal and vertical layout.

Project 2

Sliding Puzzle Game

A sliding puzzle game, with the option of either 3x3, 6x6, or 9x9 puzzles. The user would be able to select either the default, or a pre-existing photo, or a new photo as a background for the puzzle. This also employed a simple scoring and leaderboard system. My version may be found on Google Play as "SlidingFrenzy".
