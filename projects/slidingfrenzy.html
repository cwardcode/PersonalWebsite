Sliding Frenzy

An Android-OS sliding puzzle game.

About Sliding Frenzy

Sliding Frenzy was one of my first Android-based application, originally created for my Mobile Application Development course. . It allows players to select one of three different sized puzzles. Player may choose from either: 3x3, 6x6, or 9x9. Once chosen, the player may choose to either use the default image, a new image, or a pre-existing image from their gallery. Once the game starts, points are deducted from the overall score. If the player completes the puzzle, they are asked if they would want their score posted to the leaderboard.

Scoreboard Integration

The scoreboard integration of this game was an interesting feature to integrate into the application. When the user visits the leaderboard, the current board is parsed from a file located on an external server. To upload the new score, there is a server-side script that takes the users score, and appends it to the server-side document.
